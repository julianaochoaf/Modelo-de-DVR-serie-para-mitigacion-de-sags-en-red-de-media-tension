# Modelo-de-DVR-serie-para-mitigacion-de-sags-en-red-de-media-tension
Modelo de subestación AT/MT y sus redes de salida 34,5 kV, con un DVR en conexión serie estudiado para la mitigación de hundimientos de tensión sag. 

En el presente repositorio se encontrará modelo realizado en el software Simulink Matlab utilizando el toolbox Simscape Electrical, en el cual se presenta un sistema de simulación sin compensación y otro con compensación en el que se presentan los esquemas de control de inversores y convertidores utilizados.
